# ProspectIQ AI - Cognitive Inference Engines

The FastAPI AI service exposes a suite of cognitive engines under `engines/`:

1. **TrustLayer (`engines/trustlayer/`)**: Assesses customer data quality and validation scores.
2. **BehaviorIQ (`engines/behavioriq/`)**: Analyzes digital velocity, income stability, and expense disciplines.
3. **Financial DNA (`engines/findna/`)**: Synthesizes structured profile vectors.
4. **PriorityIQ (`engines/priorityiq/`)**: Calculates prospect value and relationship priority.
5. **Relationship Co-Pilot (`engines/copilot/`)**: Renders contextual conversation tips.
6. **NBAIQ (`engines/nbaiq/`)**: Evaluates Next Best Actions.
7. **ExplainIQ (`engines/explainiq/`)**: Provides decision matrix explainability evidence.
8. **PredictIQ (`engines/predictiq/`)**: Generates growth, risk, and churn predictions.
9. **PortfolioIQ (`engines/portfolioiq/`)**: Compiles aggregate manager analytics.
10. **SimulationIQ (`engines/simulationiq/`)**: Projects financial adjustment outcomes.
