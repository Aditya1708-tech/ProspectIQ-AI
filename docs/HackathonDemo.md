# IDBI Bank Innovate - Hackathon Demo Guide

A complete workflow walkthrough to demonstrate ProspectIQ AI during the Hackathon:

1. **Manager Command Center**:
   - Log in using quick access buttons.
   - Show Branch Health indicators in official colors.
2. **Customer Intelligence Foundation**:
   - Navigate to Customer List.
   - Open a detailed customer profile (e.g. Priyanshu Sharma).
3. **Cognitive Analysis Showcase**:
   - Highlight the **Trust Quality Index** gauge.
   - Run a **Simulation Scenario** (adjusting loan rates) and show projected KPI changes on the graph.
